# 정보보호 기본 개념

## 1. 정보보호의 목표 (CIA + 2)

### 1) 기밀성 (Confidentiality)
허가된 사용자만 정보에 접근할 수 있도록 하는 것

**관련 개념**
- 알 필요성 (Need to Know)
- 최소 권한 (Least Privilege)

**대표 보안 기술**
- 접근제어
- 암호화

---

### 2) 무결성 (Integrity)
데이터가 허가되지 않은 방식으로 변경되지 않도록 보호

**핵심 키워드**
- 정확성
- 변경 방지

**대표 보안 기술**
- 접근제어
- 메시지 인증
- 침입 탐지
- 백업

---

### 3) 가용성 (Availability)
권한이 있는 사용자가 필요할 때 정보 서비스를 사용할 수 있어야 함

**대표 보안 기술**
- 백업
- 시스템 중복성
- BCP / DRP

---

### 4) 인증성 (Authenticity)
사용자나 시스템이 진짜인지 확인

예시
- 로그인
- 전자서명

---

### 5) 책임추적성 (Accountability)
누가 어떤 행동을 했는지 추적 가능해야 함

관련 기술
- 로그 기록
- 포렌식
- 부인방지

---

# 2. 정보보호 관리

정보보호 대책은 다음 3가지로 분류

```
관리적 보호대책
물리적 보호대책
기술적 보호대책
```

---

## 1) 관리적 보호대책
정책, 제도, 절차 등 **조직 관리 중심 보안**

예시
- 보안 정책
- 보안 교육
- 보안 절차

---

## 2) 물리적 보호대책
시설과 장비 보호

예시
- 출입 통제
- 시건(잠금)
- CCTV

자연재해 대응

- 화재
- 수해
- 지진
- 태풍

---

## 3) 기술적 보호대책
기술을 이용한 보안 대책

예시
- 접근 통제
- 암호 기술
- 백업 체계

---

# 3. OSI 보안 구조 (X.800)

ITU(International Telecommunication Union)에서 제정한 보안 구조

보안은 다음 3가지로 분류

```
보안 공격 (Security Attack)
보안 메커니즘 (Security Mechanism)
보안 서비스 (Security Service)
```

---

## 보안 공격
정보 시스템을 침해하는 행위

---

## 보안 메커니즘
보안 공격을 탐지하거나 예방하는 기술

---

## 보안 서비스
보안 메커니즘을 이용하여 제공되는 보안 기능

예시

- 기밀성
- 무결성
- 가용성
- 인증
- 부인방지
- 접근제어

---

# 4. 보안 공격 유형

## 4.1 소극적 공격 (Passive Attack)

시스템에 영향을 주지 않고 **정보만 수집하는 공격**

특징
- 시스템 변화 없음
- 탐지 어려움
- 예방 중요

공격 유형

```
Snooping (스누핑)
Traffic Analysis (트래픽 분석)
```

---

### Snooping (스누핑)
데이터를 몰래 가로채는 공격

예시
- 네트워크 패킷 감청

대응 방법
- 암호화

---

### Traffic Analysis (트래픽 분석)
통신 내용은 보지 못해도 **통신 패턴을 분석하는 공격**

예시
- 누가 누구에게 얼마나 자주 통신하는지 분석

---

# 4.2 적극적 공격 (Active Attack)

시스템이나 데이터를 직접 변경하거나 공격

특징
- 시스템 영향 있음
- 탐지 가능

공격 유형

```
Modification (변경)
Masquerading (가장)
Replay (재연)
Repudiation (부인)
Denial of Service (DoS)
```

---

### Modification (변경)
데이터를 불법적으로 수정하거나 메시지 순서를 변경

---

### Masquerading (가장)
다른 사용자나 시스템으로 신분 위장

---

### Replay (재연)
패킷을 저장했다가 다시 전송

---

### Repudiation (부인)
자신이 수행한 행동을 부인

예시

- 송신 부인
- 수신 부인

---

### Denial of Service (DoS)
서비스를 지연시키거나 마비시키는 공격

예시
- Slowloris

---

# 5. 소극적 공격 vs 적극적 공격

| 구분 | 특징 | 공격 |
|---|---|---|
| 소극적 공격 | 정보만 수집 | 스누핑, 트래픽 분석 |
| 적극적 공격 | 시스템 영향 | 변경, 가장, 재연, 부인, DoS |
