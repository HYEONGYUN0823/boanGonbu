# 주요 암호 기술 개요

# 1. 대칭키 암호 vs 비대칭키 암호

## 대칭키 암호 (Symmetric Key)

특징

- 암호화 키 = 복호화 키
- 처리 속도가 빠름
- 일반 데이터 암호화에 사용

장점

- 빠른 처리 속도
- 작은 키 길이

단점

- 키 분배 문제 발생

예시

```
AES
DES
```

---

## 비대칭키 암호 (Public Key Cryptography)

특징

- 서로 다른 키 사용

```
공개키
개인키
```

특징

- 처리 속도 느림
- 작은 데이터 암호화에 사용

주요 사용

```
키 분배
전자서명
```

---

# 2. 하이브리드 암호 시스템

대칭키와 공개키의 **장점을 결합한 방식**

동작 방식

```
1. 공개키로 세션키 전달
2. 세션키로 실제 데이터 암호화
```

예시

```
HTTPS
SSL/TLS
```

---

# 3. 일방향 해시 함수 (Hash Function)

특징

```
임의 길이 입력 → 고정 길이 출력
```

특징

- 복호화 불가능
- 입력값 유추 불가능

주요 사용

```
비밀번호 저장
무결성 검증
```

예시

```
SHA-256
SHA-512
```

---

# 4. 메시지 인증 코드 (MAC)

제공 기능

```
무결성
인증
```

메시지가 **변조되지 않았는지 확인**

---

# 5. 전자서명 (Digital Signature)

제공 기능

```
무결성
인증
부인방지
```

사용 예

```
전자 계약
전자 문서
```

---

# 암호 분석 (Cryptanalysis)

## 개요

암호문으로부터

```
평문
또는
키
```

를 찾는 과정

---

## 복호화 vs 해독

| 구분 | 의미 |
|---|---|
| 복호화 | 정상 사용자가 키로 평문 복원 |
| 해독 | 공격자가 키 없이 암호 해석 |

---

# 케르히호프의 원리

암호 시스템의 안전성은

```
알고리즘이 아니라
키의 비밀에 의존해야 한다
```

즉

```
알고리즘 공개
키는 비밀
```

---

# 암호 공격 방법

## 1. 암호문 단독 공격 (COA)

공격자가 가진 것

```
암호문만
```

---

## 2. 기지 평문 공격 (KPA)

공격자가 가진 것

```
평문 + 암호문 쌍
```

---

## 3. 선택 평문 공격 (CPA)

공격자가

```
평문을 선택하여 암호화
```

능동적 공격

---

## 4. 선택 암호문 공격 (CCA)

공격자가

```
암호문을 선택하여 복호화 결과 획득
```

가장 강력한 공격

---

## 공격 강도

```
COA < KPA < CPA < CCA
```

---

# 특이한 공격

## 고무호스 암호분석

협박이나 고문을 통해 **키를 알아내는 공격**

가장 현실적인 공격

---

# 암호 강도를 결정하는 요소

## 대칭키 암호

```
키 길이
라운드 수
블록 크기
```

---

## 비대칭키 암호

```
수학적 난이도
```

예시

```
소인수 분해
이산대수 문제
```

---

## 해시 함수

```
해시값 길이
```

권장

```
SHA-256
SHA-512
```

---

# 지적 재산권 보호 기술

## 스테가노그래피 (Steganography)

메시지를 **숨기는 기술**

예시

```
이미지 파일에 정보 숨김
```

핵심

```
은닉
```

---

## 디지털 워터마킹 (Digital Watermark)

콘텐츠에

```
저작권 정보 삽입
```

예시

```
지폐
```

종류

```
강한 워터마킹
약한 워터마킹
```

---

## 핑거프린팅 (Fingerprinting)

콘텐츠에

```
구매자 정보 삽입
```

목적

```
불법 유통 추적
```

---

## 핵심 차이

```
스테가노그래피 → 숨김
워터마킹 → 저작권 표시
핑거프린팅 → 구매자 추적
```

---

# DRM (Digital Rights Management)

디지털 콘텐츠의

```
사용 권한
유통
과금
```

을 관리하는 기술

---

## DRM 구성 요소

```
메타데이터
패키저
시큐어 컨테이너
식별자
DRM 제어기
```

---

## DRM 구성 주체

| 구성 | 역할 |
|---|---|
| 콘텐츠 제공자 | 콘텐츠 제작 |
| 콘텐츠 배포자 | 콘텐츠 유통 |
| 콘텐츠 소비자 | 콘텐츠 사용 |
| 클리어링하우스 | 결제 및 사용 기록 관리 |

---
