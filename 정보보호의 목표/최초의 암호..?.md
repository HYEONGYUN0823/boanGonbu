# 최초의 전치암호 : scytale
## 전치암호: 문자 순서를 재배열하는 방식

✔ 고대 스파르타에서 사용
✔ 막대기에 종이를 감아서 암호 작성
✔ 문자 위치를 바꿈 (순서 변경)
✔ 문자는 그대로

👉 그래서 전치(Transposition) 암호

예:

HELLOWORLD

를 일정 간격으로 재배열

핵심:
문자는 안 바뀌고 자리만 바뀜

# 최초의 치환 암호 : 시저
## 치환암호: 문자를 다른 문자로 교체하는 방식

✔ 로마의 Julius Caesar 사용
✔ 알파벳을 일정 칸 밀어서 바꿈
✔ 문자를 다른 문자로 교체

예:

A → D
B → E
C → F

HELLO → KHOOR

핵심:
문자 자체를 다른 문자로 바꿈
