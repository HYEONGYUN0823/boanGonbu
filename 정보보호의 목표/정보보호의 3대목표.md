# 정보보호의 목표
## 기밀성(비밀성)(C)
몰래 못보기

한문장 : 허가된 사람만 볼 수 있게 하는 것

## 무결성(I)
몰래 못 바꾸기

한문장 : 데이터가 몰래 바뀌지 않았다는 보장

## 가용성 (A)
필요할때 쓰기

한문장 : 필요할 때 정상적으로 사용할 수 있는 상태

## 인증 (진짜인지 아닌지)
사용자 인증, 메시지 인증

## 부인방지(오리발방지)
예시로 전자서명같은거 생각하면 될듯 (인증, 부인방지)
