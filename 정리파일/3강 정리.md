# 암호학 개요
## 암호학(Cryptology)의 정의
암호학(Cryptology)이란 : 암호기술(cryptography), 암호해독(cryptanalysis)
## 암호에서 사용하는 이름
앨리스, 밥, 이브, 맬로리, 트렌트, 빅터

앨리스와 밥 : 앨리스는 메시지를 전송하고 밥이 수신을 하는 모델에 사용

이브 : 소극적인 공격자

맬로리 : 악의를 가진(malicious) 공격자를 의미, 적극적공격
