# 키
## 개요
1. 키 공간(Key_Space)의 크기, 즉 가능한 키의 총 수는 크면 클수록 [무차별] 공격에 강하다. 키 공간의 크기는 키의 비트 길이로 정해진다.
2. 키는 평문과 같은 가치를 갖는다 (암호해독 목적 : 키, 평문)
# 다양한 키 
## 세션키와 마스터 키 *3
1. 통신 때마다 한 번만 사용되는 키를 [세션키]라고 한다
2. 반복적으로 사용되는 키를 [마스터 키]라고 부른다

## CEK(세션키와 KEK(마스터 키)
1. 통상적으로 사용자가 직접 이용하는 정보가 암호홛의 대상이 된다. 이때 사용하는 키를 CEK라 불느다
2. 이와는 달리 키를 암호화하는 키를 KEK라 부른다.

# 패스워드를 기초로 한 암호(PBE)
## 솔트의 역할 *5
1. 솔트는 사전공격 또는 [레인보우] 테이블을 이용한 공격을 [막기] 위해 존재한다.

## PBKDF
1. PBKDF은 비밀번호, 반복횟수, 생성하고자 하는 비밀키의 [길이]를 입력으로 [비밀키]를 [출력]한다.

## 암호키 저장 방법의 예 *3
1. 하드웨어 토큰은 저장도니 정보가 위*변조 또는 외부로 노출되기 어령누 장치로 스마트카드, USB 도큰 등의 [보안토큰(HSM, Hardware Security Module)]을 의미한다.

# 난수
## 난수의 용도
* 키의 생성 : 대칭키 암호나 메시지 인증 코드에서 사용한다.
*  키쌍의 생성 : 공개키 암호나 전자서명에서 사용한다.
*  초기화 벡트(IV)의 생성 : 블록 암호 모드인 CBC, CFB, OFB에 각ㄱ각 사용한다
*  비표(nonce)의 생성 : 재전송 공격방지나 블록 암호의 CTR 모드 등에서 사용한다.
*  슬트의 생성 : 패스워드를 기초로 한 암호화(PBE) 등에서 사용한다.
*  일회용 패드 : 패딩에 사용되는 열을 생성하는데 사용한다.

# 난수의 성질
## 난수의 성질 분류
* 무작위성 : 통계적인 편중 없이 수열이 무작위로 되어 있는 성질
* 예측 불가능성 : 과거의 수열로부터 다음 수를 예측할 수 없다는 성질
* 재현 불가능성 : 같은 수열을 재현할 수 없다는 성질. 재현하기 위해서는 수열 그 가체를 [보존]해두는 수 밖에 없다.

# 의사난수 생성기 *3
## 개요
1. 하드웨어를 난수생성기(RNG)라 부른다.
2. 소프트웨어를 의사난수 생성기(PRNG)라 부른다. 소프트웨어만으로는 진정한 난수를 생성할 수 없기 떄문에 [의사]난수 생성기라 불느다
## 의사난수 생성기의 구조
1. 의사난수 생성기는 내부상태를 가지며, 외부에서 주어진 [종자(seed)](랜덤비트열(비밀))를 기초로 의사난수열을 생성한다. **내부상태**라고 하는 것은 요컨대 의사난수 생성기가 관리하고 있는 [메모리 값]을 말한다.
